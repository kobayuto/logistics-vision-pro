# Logistics Vision Pro for Windows

ローカルのExcel/CSVから倉庫のレイアウト・動線・ABC・在庫適合をオフラインで分析する、Windowsデスクトップアプリです。GAS版と同じ画面・同じ結果。インターネット不要・GitHubアカウント不要・無料。

> このリポジトリは **配布専用** です（ソースは非公開）。ビルド成果物のみを公開しています。

## ダウンロード

- 📥 **[最新版インストーラ](https://github.com/kobayuto/logistics-vision-pro/releases/latest/download/LogisticsVisionPro-Windows-Setup.exe)**（`-setup.exe`）
- または [リリース一覧](https://github.com/kobayuto/logistics-vision-pro/releases)

紹介ページ（ダウンロード・手順）: GitHub Pages（このリポジトリの Pages）。

## インストール手順

1. ダウンロードした `…-Setup.exe` を実行。
2. 「**WindowsによってPCが保護されました**」(SmartScreen) が出たら → **詳細情報** → **実行**。
   - ※ コード署名前のため警告が出ますが、アプリは外部に一切通信しません。
3. 起動 → ワークスペースフォルダ（データ/設定の置き場）を選択して完了。

## 動作要件

- Windows 10 / 11 (x64)
- インターネット不要（WebView2 同梱）
- データはローカルのみ（外部送信なし）
