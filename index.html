Install

インストール手順

1

ダウンロード

上の「ダウンロード」から …-Setup.exe を取得し、実行します。

2

SmartScreen警告が出たら

「WindowsによってPCが保護されました」が表示されたら、詳細情報 をクリック → 実行 を選びます。
※ 署名前のため警告が出ますが、このアプリは外部に一切通信しません。

3

起動して使う

インストール後にアプリを起動 → ワークスペースフォルダ（データ/設定を置く場所）を選択すれば準備完了です。

なぜ警告？ 配布コストを抑えるためコード署名をしていないためです（既知・想定内）。インストーラはWebView2を同梱しており、インターネットの無いPCでも動作します。アプリは外部送信を行いません。

Features

GAS版と同一エンジン。ローカルでそのまま。

🗺️

レイアウト可視化

ロケコードから倉庫フロアマップを生成。ゾーン・通路・段/間口まで再現。

🚶

ピッキング動線

実績からピッカー動線・混雑を可視化。通路ネットワーク上で経路計算。

📊

ABC / 在庫適合

出荷からABC（SABC/ABCD）、在庫の占有・適合をゾーン別に分析。

🔁

在庫移動分析

移動実績からペア・ゾーン間移動を集計。

🔎

在庫検索

品番/品名でロケ横断検索し、該当ロケをマップ上でハイライト。

🔒

互換検証済み

解析エンジンはGAS版とゴールデン照合で一致を機械検証。結果は同一。
